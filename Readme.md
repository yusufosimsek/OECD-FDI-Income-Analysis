# OECD FDI Income Analysis by Economic Activity

This project aims to analyze sector-based **Foreign Direct Investment (FDI) income** of countries using an OECD dataset and explore patterns and relationships within the data.

## 📊 Project Overview
The dataset consists of **24 variables and 10,344 observations**. Within the scope of this project, the data will undergo **data cleaning, exploratory data analysis (EDA), and statistical analysis** in order to better understand the distribution and relationships of FDI income across different economic sectors.

## 🛠️ Technologies Used
- **Language:** Python (Jupyter Notebook)
- **Libraries:** Pandas, NumPy, Seaborn, Matplotlib
- **Reporting & Visualization:** Power BI

## 📑 Analysis Workflow
The planned analysis steps are as follows:
- **Data Understanding:** Identifying variable types and generating descriptive statistics.
- **Data Cleaning:** Handling missing values, inconsistencies, and formatting issues.
- **Distribution Analysis:** Examining numerical variables with statistical summaries and visualizations.
- **Relationship Analysis:** Investigating dependencies between variables using statistical tests and correlation analysis.
- **Reporting:** Presenting insights through visualizations and dashboards in Power BI.
